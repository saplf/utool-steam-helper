# Steam 助手

## 功能

- 使用 `sgame` 获取本机已安装 steam 游戏，回车后通过 steam 打开游戏

- 使用 `sseting` 打开设置，目前只支持切换显示用户（用于显示列表中的游玩时间）

## 注意事项

- 目前只支持 win & mac（目前无 linux 环境，无法进行开发与测试）

- win 下，steam 的安装路径是通过注册表查询的，所以绿色版目前无法获取游戏目录

- mac 下，steam 目录是固定的，后续考虑支持指定 steam 目录
